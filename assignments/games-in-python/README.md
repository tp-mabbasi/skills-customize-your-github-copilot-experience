# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game in Python using strings, loops, and conditionals. By completing this assignment, students will practice game logic, user input handling, and random word selection.

## 📝 Tasks

### 🛠️	Set Up the Game and Select a Word

#### Description
Create the initial game setup for Hangman. Define a list of possible words and randomly choose one word for the current game.

#### Requirements
Completed program should:

- Store multiple possible words in a predefined list.
- Randomly select one word from the list at the start of the game.
- Display the hidden word as underscores separated by spaces (for example: _ _ _ _).


### 🛠️	Implement the Guessing Loop and Win/Lose Logic

#### Description
Build the main gameplay loop where the player enters letter guesses until they either reveal the full word or run out of attempts.

#### Requirements
Completed program should:

- Accept one-letter guesses from the player and update progress when guesses are correct.
- Track and display remaining incorrect attempts after wrong guesses.
- End the game with a clear win message when the word is guessed or a lose message when attempts reach zero.
